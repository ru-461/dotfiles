# dotfiles

My settings and any more dotfiles.
Configuration scripts that can be used anywhere.

# Support OS

- macOS（only arm64）
- WSL2 （only Ubuntu x64）
- Ubuntu（x64）
- Linux（x64）

# How to use

Run bootstrap.sh !
Just run this one-liner！

```shell
bash <(curl -fsSL raw.githubusercontent.com/ryu-461/dotfiles/main/bootstrap.sh)
```

Happy Hacking!!
