# dotfiles

My settings and any more.

## Support OS

- macOS（only arm64）
- WSL2 （Ubuntu x64）
- Ubuntu（x64）
- Linux（x64）

## How to use

### macOS

```shell
# Install xcode-select
xcode-select install

# Clone this repository locally
git clone https://github.com/ryu-461/dotfiles.git
cd dotfiles

# Run bootstrap.sh
 ./bootstrap.sh
```

### Ubuntu・WSL

```shell
# Run bootstrap scrpt
bash <(curl -sL raw.github.com/ryu-461/dotfiles/bootstrap.sh)
```

### Linux

```shell
# Run bootstrap scrpt
bash <(curl -sL raw.github.com/ryu-461/dotfiles/bootstrap.sh)
```
